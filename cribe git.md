# Шпаргалка по git

## Команды

*git version* - команда, отображающая текущую версию git.

*git status* - команда, отображающая состояние файла.

*git init* - команда, инициализируащая локальный репоиторий.

*git add + название файла с расширением*

*git commit -m "text"*

*git diff*

*git log*

*git checkout*

*git checkout master*

*clear*
